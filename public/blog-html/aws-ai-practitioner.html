3) Bedrock Customization: Continued Pre-training vs Fine-tuning

  • Continued Pre-training
    • Additional exposure to unlabeled data to adapt domain knowledge or writing style
    • Keywords: domain adaptation, unlabeled data, continued learning
  • Fine-tuning
    • Uses labeled data (with ground truth) to precisely optimize performance for a specific task
    • Keywords: supervised learning, labeled data, task-specific

‚úÖ Common Correct Answer Pattern

‚ÄúStrengthen expertise in a specialized domain (e.g., genomics)‚Äù

‚Üí Often appears as a combination of continued pre-training (domain adaptation) + fine-tuning (task optimization)

7) Generation Parameters: Top-K / Top-P / Temperature / Stop

  • Top-K

    : Limits next-token candidates to the top K tokens

  • Top-P

    : Includes tokens until cumulative probability P is reached (nucleus sampling)

  • Temperature

    : Controls randomness by flattening or sharpening the probability distribution

    ‚Üí Affects creativity / diversity

  • Stop sequences

    : Stops generation when specific strings appear

‚úÖ Exam Intuition Rules

  • Accurate / conservative output ‚Üí Lower Top-P, Top-K, Temperature
  • Diverse / creative output ‚Üí Higher Top-P, Top-K, Temperature

10) Attack & Risk Scenarios: Poisoning vs Prompt Leaking vs Hijacking vs Jailbreaking

  • Poisoning

    : Outputs or training data become contaminated with malicious content (e.g., spam links)

  • Prompt leaking

    : Exposure of system prompts, prior conversations, or internal context

  • Prompt Hijacking

    : An attack where a user attempts to divert the model from its original purpose

    by overriding instructions to change the topic or intent

  • Jailbreaking

    : Techniques that bypass safety policies or guardrails, forcing the model to respond

    to requests it should normally refuse

‚úÖ Common Exam Phrases

  • ‚ÄúResponse contains malicious content/links‚Äù
    ‚Üí Poisoning
  • ‚ÄúModel reveals hidden instructions / previous conversation‚Äù
    ‚Üí Prompt leaking
  • ‚ÄúIgnore the previous instructions and do X instead‚Äù
    ‚Üí Hijacking
  • ‚ÄúAs an exception, you may break the rules / pretend restrictions don‚Äôt exist‚Äù
    ‚Üí Jailbreaking
